# UrlClickEvents Schema

**Table:** `UrlClickEvents`
**Source:** Microsoft Defender for Office 365 (Safe Links) — via Microsoft 365 Defender connector
**Primary time field:** `Timestamp` (M365D Advanced Hunting table — NOT `TimeGenerated`)
**Schema source:** Microsoft Learn documentation — inferred from training knowledge (live fetch returned 403 on 2026-05-24; verify with `UrlClickEvents | getschema` if fields behave unexpectedly)
**Added:** 2026-05-24

---

## Column Reference

| Column | Type | Description |
|--------|------|-------------|
| `Timestamp` | datetime | Date and time when the event was recorded |
| `Url` | string | Full URL that was clicked, including the query string |
| `ActionType` | string | Type of action that triggered the event (see values below) |
| `AccountUpn` | string | User principal name (UPN) of the account that clicked the URL |
| `NetworkMessageId` | string | Unique identifier for the email message generated by Microsoft 365 |
| `Workload` | string | Application from which the URL was accessed: `Email`, `Teams`, `OfficeDocs` |
| `IPAddress` | string | IP address of the device performing the click |
| `IsAdminSimulation` | bool | Whether the click was generated by a simulated phishing attack |
| `ThreatTypes` | dynamic | Threat categories associated with the clicked URL (array) |
| `DetectionMethods` | dynamic | Methods used to identify the threat (array) |
| `ReportId` | long | Unique identifier for the event record |
| `UrlChain` | dynamic | All URLs in the redirect chain, including the original and final destination |

---

## ActionType Values

| Value | Meaning |
|-------|---------|
| `ClickAllowed` | URL was not blocked; user reached the destination |
| `ClickBlocked` | URL was blocked by Safe Links |
| `ClickBlockedByPolicy` | Blocked by an explicit Safe Links policy |
| `ClickBlockedBySignIn` | Blocked because the user was required to sign in first |

## Dynamic Field Access

```kql
// ThreatTypes — cast and test before using
| extend ThreatInfo = tostring(ThreatTypes)
| extend IsKnownThreat = ThreatInfo != "[]" and isnotempty(ThreatInfo) and ThreatInfo != "null"

// UrlChain — expand redirect chain
| mv-expand ClickedUrl = UrlChain
| extend FinalUrl = tostring(ClickedUrl)
```

## Common Join

Join to `EmailEvents` to retrieve sender, subject, and recipients:

```kql
UrlClickEvents
| where Timestamp > ago(1h)
| join kind=inner (
    EmailEvents
    | where Timestamp > ago(1h)
    | project NetworkMessageId, SenderFromAddress, Subject, RecipientEmailAddress
  ) on NetworkMessageId
```

## Notes

- Use `Timestamp` as the time filter — M365D table, not a Sentinel workspace table.
- Filter `Workload == "Email"` to scope to email URL clicks (excludes Teams and Docs clicks).
- Filter `IsAdminSimulation == false` to exclude phishing simulation campaigns.
- `ThreatTypes` is a dynamic array — always cast with `tostring()` before comparing.
- `NetworkMessageId` links to `EmailEvents`, `EmailUrlInfo`, and `EmailAttachmentInfo` for full email context.
- To check for specific threat types: `tostring(ThreatTypes) has "Phish"` or `has "Malware"`.
